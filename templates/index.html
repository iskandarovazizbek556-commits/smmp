SMMZone.uz xavfsizmi? Accountim bloklanadimi?
Bizning barcha xizmatlarimiz Instagram, YouTube va boshqa platformalarning qoidalariga mos ravishda ishlaydi. Biz faqat real va faol profillarga asoslangan xizmatlar taqdim etamiz. Shunga qaramay, bir kunda juda ko'p miqdorda buyurtma berishdan saqlaning — bu har qanday SMM panelida xavf tug'dirishi mumkin.
Buyurtma bergandan keyin qancha vaqtda boshlanadi?
Odatda buyurtmangiz 1–5 daqiqa ichida avtomatik ravishda boshlanadi. Yuklanish vaqtida ba'zida kechikish bo'lishi mumkin, lekin 15 daqiqadan oshmasligi kerak. Agar 15 daqiqadan ortiq kutgan bo'lsangiz, support bilan bog'laning.
Balansni qanday to'ldirish mumkin?
Payme, Click, Uzcard, Humo yoki USDT orqali balans to'ldirishingiz mumkin. To'lov darhol qayta ishlanadi va balans 1–2 daqiqa ichida yoziladi. Minimal to'ldirish miqdori 5 000 so'm.
API bor va u qanday ishlaydi?
Ha, biz to'liq REST API taqdim etamiz. Reseller bo'lsangiz, API orqali o'z panelingizni qurishingiz mumkin. API kaliti shaxsiy kabinetingizda API bo'limida mavjud. Texnik hujjatlar ham o'sha yerda joylashgan.
Buyurtma to'liq bajarilmasa qaytarim bo'ladimi?
Agar buyurtma texnik sabablarga ko'ra to'liq bajarilmasa, bajarilmagan qism uchun balans avtomatik qaytariladi. Buni "Buyurtmalarim" bo'limidan kuzatishingiz mumkin.
Qanday qilib reseller bo'lish mumkin?
Ro'yxatdan o'tib, API kalitingizni oling. Bizning narxlarimizga ustama qo'shib o'z mijozlaringizga xizmat ko'rsatishingiz mumkin. Katta hajmli buyurtmalar uchun maxsus chegirmalar mavjud — Telegram orqali bog'laning.
Yordam kerak bo'lsa qayerga murojaat qilaman?
Telegram orqali @smmzone_uz manziliga yozing. Biz 24/7 ishlaymiz va har qanday muammoingizni hal qilishga tayyormiz. Javob vaqti odatda 5–15 daqiqa.
